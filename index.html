MetaMask Wallet Features

1. Create & Import Wallet

Users can create a new wallet secured by a seed phrase or import an existing one using their 12/24-word recovery phrase. This ensures portability and easy access across devices.

2. Secure Private Key Storage

Private keys are stored locally in the user's browser, encrypted with a password. MetaMask never uploads or accesses these keys remotely, ensuring user control.

3. Send & Receive Cryptocurrencies

Send and receive ETH and ERC-20 tokens. Transactions are signed and broadcast securely, with gas fee options (low, medium, high) based on the Ethereum network load.

4. Connect to DApps

Interact directly with decentralized apps (DApps) on Web3 websites. MetaMask injects web3 objects into the browser to enable smart contract interactions securely.

5. Switch Networks

Supports Ethereum Mainnet, testnets (Ropsten, Goerli), and custom networks like Binance Smart Chain. Users can add or switch networks manually or via prompts from DApps.

6. Manage Tokens

Users can search for, add, and organize ERC-20 tokens. MetaMask can auto-detect popular tokens, and allows manual addition using contract addresses.

7. View Transaction History

Every transaction made through the wallet is logged with timestamps, status (success/fail), and linked to a block explorer (like Etherscan) for full transparency.

8. Signing Messages

Allows users to cryptographically sign messages and data, often used for login/authentication in decentralized apps without needing passwords.

9. Swap Tokens

MetaMask includes a built-in token swap feature that lets users exchange tokens directly within the wallet using aggregators like 0x or Uniswap.

10. Hardware Wallet Support

Users can connect hardware wallets like Ledger and Trezor for additional security while using MetaMask as the interface.
